# Art Capsul

**Art Capsul** is a web-based platform dedicated to showcasing digital artwork and artist profiles through a modern, responsive interface. This project serves as a foundational application that demonstrates core front-end development skills using HTML, CSS, and JavaScript.

---

## 🚀 Project Overview

Art Capsul is designed to:

- Showcase featured artworks from various artists  
- Provide basic user interaction through Login & Signup  
- Display curated galleries and individual artist profiles  
- Deliver a clean, visually appealing, and responsive user interface  

This project is a **recreation and improvement of an earlier version**, with a stronger focus on code structure, maintainability, and overall user experience.

---

## 🎓 About This Project

This is my **first project as a web development student**, created to apply and strengthen my understanding of fundamental web technologies. It reflects my learning progress in building real-world interfaces and structuring a complete front-end application.

## 🧠 Goals

- Write clean and semantic HTML  
- Apply modern and efficient CSS practices
